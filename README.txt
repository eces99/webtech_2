# Hotel-Website README

## Beschreibung

Herzlich willkommen in der README-Datei für die Hotel-Website. Unsere Website bietet eine Vielzahl von Funktionen, darunter die Möglichkeit, Stammdaten zu ändern, Zimmer zu reservieren, News zu posten und vieles mehr.

## Funktionen

1. **Anmelden und Login:**
   - Registrieren Sie sich für ein persönliches Konto (Das wird später mit Datenbankverbindung aktiviert.) und melden Sie sich an, um personalisierte Funktionen zu nutzen.

2. **Stammdaten ändern:**
   - Als angemeldeter Benutzer können Sie Ihre Stammdaten bequem bearbeiten.

3. **Buchung:**
   - Reservieren Sie Ihr gewünschtes Zimmer nach dem Login.

4. **Reservierungen ansehen:**
   - Überprüfen Sie Ihre aktuellen und vergangenen Reservierungen.

5. **News posten (Admin-Funktion):**
   - Als Administrator können Sie Neuigkeiten auf der News-Seite veröffentlichen.
   - Die geposteten Neuigkeiten sind für alle Besucher sichtbar.

6. **Footer und Impressum:**
   - Der Footer enthält grundlegende Informationen und einen Link zum Impressum.

## Installation

1. Laden Sie die Dateien aus dem Repository herunter.
2. Stellen Sie sicher, dass ein Webserver (z.B. Apache) und PHP installiert sind.
3. Kopieren Sie die Dateien in das Webserver-Root-Verzeichnis.

## Nutzung

1. Öffnen Sie den Webbrowser und navigieren Sie zur Website.
2. Registrieren Sie sich oder melden Sie sich an.
3. Ändern Sie bei Bedarf Ihre Stammdaten.
4. Nach dem Login können Sie Zimmer buchen, Ihre Reservierungen einsehen und News posten (als Administrator).
5. Überprüfen Sie den Footer für zusätzliche Informationen und den Link zum Impressum.
